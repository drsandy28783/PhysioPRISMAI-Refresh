## Production Notes for All Videos

### General Guidelines:

**Visual Style:**
- Clean, modern interface shots
- Screen recordings with cursor animations
- Text overlays in sans-serif font (Arial/Helvetica)
- Brand colors: Teal (#1a5f5a), White, Gray

**Audio:**
- Upbeat background music (no lyrics)
- Professional voiceover OR on-screen text only
- Sound effects: Click sounds, success chimes, sparkle sounds

**Pacing:**
- Fast cuts (2-3 seconds per shot max)
- Quick transitions (fade or cut, no fancy effects)
- Maintain energy throughout

**Text Overlay Style:**
- Large, bold font
- High contrast (white text on dark background or vice versa)
- Appear with subtle animation (slide in from side)
- Stay on screen long enough to read (3-5 seconds)

### Equipment Needed:

1. Screen recording software (OBS, Loom, or ScreenFlow)
2. Video editing software (CapCut, Adobe Premiere, Final Cut Pro)
3. Cursor animation tool (for professional cursor movements)
4. Text animation templates
5. Royalty-free music library

### Recording Tips:

1. **Screen Recording:**
   - Record at 1080p minimum
   - Use cursor highlight tool
   - Slow down cursor movements for clarity
   - Pause before clicking for emphasis

2. **Editing:**
   - Speed up typing/loading to keep pace
   - Add zoom-in effects on important text
   - Use highlight boxes around AI suggestions
   - Add subtle screen shake on "wow" moments

3. **Text Overlays:**
   - Keep text short (max 7 words per line)
   - Use emojis sparingly for emphasis
   - Make CTAs stand out (different color/size)

### Platform Optimization:

**Instagram Reels:**
- 9:16 vertical format
- 15-30 seconds ideal
- Front-load the hook (first 3 seconds critical)
- Use all 30 hashtags in caption

**TikTok:**
- 9:16 vertical format
- 15-60 seconds
- On-screen text preferred (many watch without sound)
- Trending audio can boost reach

**YouTube Shorts:**
- 9:16 vertical format
- Up to 60 seconds
- Add strong verbal CTA at end
- Link in description and pinned comment

**LinkedIn:**
- Can be horizontal (16:9) or vertical (9:16)
- 30-60 seconds ideal for B2B
- Professional tone
- Subtitle recommended (auto-play without sound)

## Tracking Metrics

### Key Performance Indicators:

**Engagement:**
- Views (aim for >1000 in first 24 hours)
- Completion rate (>50% watch to end)
- Likes, comments, shares
- Saves (high intent indicator)

**Traffic:**
- Link clicks (bio link or swipe-up)
- Profile visits
- Website traffic from social

**Conversion:**
- Trial signups from social
- "How did you hear about us?" responses
- Promo code usage if applicable

### Optimization Actions:

**If views are low:**
- Test different hooks
- Post at different times
- Try trending audio (TikTok/Reels)
- Engage with others before posting

**If engagement is low:**
- Shorten video (aim for 15-20s)
- Front-load value
- Stronger CTA
- More relatable pain point

**If clicks are low:**
- Clearer CTA
- Stronger value proposition
- Add urgency ("Limited trial spots")
