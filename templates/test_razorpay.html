Razorpay Test Mode Setup Checklist

  • Test API Keys: Use keys starting with rzp_test_
  • Create Plans in Razorpay Dashboard:
    1. Go to Razorpay Subscriptions
    2. Create plans for: Starter, Professional, Clinic, Institute (5, 10, 15 users)
    3. Note down each Plan ID (starts with plan_)
  • Set Environment Variables: 
    RAZORPAY_KEY_ID=rzp_test_xxxxxxxxxx
RAZORPAY_KEY_SECRET=your_test_secret
RAZORPAY_WEBHOOK_SECRET=your_webhook_secret
RAZORPAY_PLAN_STARTER=plan_xxxxx
RAZORPAY_PLAN_PROFESSIONAL=plan_xxxxx
RAZORPAY_PLAN_CLINIC=plan_xxxxx
RAZORPAY_PLAN_INSTITUTE_5=plan_xxxxx
RAZORPAY_PLAN_INSTITUTE_10=plan_xxxxx
RAZORPAY_PLAN_INSTITUTE_15=plan_xxxxx
