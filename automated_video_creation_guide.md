## 💡 Pro Tips for InVideo Success

### 1. Template Consistency
- Create ONE master template for all videos
- Same intro/outro style
- Consistent fonts, colors, transitions
- Brand recognition

### 2. Voiceover Strategy
**Option A:** Use InVideo AI voices
- Consistent across all videos
- Professional quality
- No recording needed
- **Recommended voices:** "Joanna" (female), "Matthew" (male)

**Option B:** Record your own voice
- More authentic
- Use Descript for voice cloning
- Can create unlimited variations
- Personal connection with audience

### 3. Music Library
- Choose 3-5 tracks from InVideo library
- Use same track family across related videos
- Keep volume at 20-30% (don't overpower voiceover)
- Avoid music with lyrics (distracting)

### 4. Text Overlay Best Practices
- Maximum 7 words per line
- Use high contrast (white on dark or vice versa)
- Add subtle drop shadow for readability
- Animate in from side or fade (not bouncy effects)
- Stay on screen 3-5 seconds minimum

### 5. Export Settings
**Instagram Reels:**
- 9:16 aspect ratio
- 1080 x 1920 resolution
- 30 fps
- MP4 format
- Under 60 seconds

**TikTok:**
- Same as Reels
- Can be up to 3 minutes but 15-60s performs best

**LinkedIn:**
- Can use 9:16 or 16:9
- 1080p minimum
- Add captions (auto-play without sound)

## 🆘 Troubleshooting Common Issues

### "InVideo AI doesn't understand my app"
**Solution:** Be very specific in prompts, reference exact UI elements
- Instead of: "Show the form"
- Use: "Show a teal-colored web form with fields labeled 'Goals (Patient-Centric)', 'Baseline Status', 'Measurable Outcomes' on a light background"

### "AI voice sounds robotic"
**Solution:**
- Use InVideo's premium voices (Joanna, Matthew work well)
- Adjust speed to 0.95x (slightly slower sounds more natural)
- Add pauses with punctuation (... or —)
- OR use your own voice recording

### "Videos too long for platform"
**Solution:**
- Each platform has different sweet spots
- Instagram Reels: 15-30s (best engagement)
- TikTok: 15-60s
- LinkedIn: 30-60s
- Trim ruthlessly - hook in first 3s or lose viewers

### "Screen recordings look blurry"
**Solution:**
- Record at 1920x1080 minimum
- Don't scale up smaller recordings
- Use "Retina" or "HiDPI" mode if available
- InVideo can upscale but native quality is better
